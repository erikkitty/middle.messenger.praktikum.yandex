# Chat Application

Мессенджер с возможностью обмена сообщениями, управления профилем и настройками пользователя.

## Деплой

Проект развёрнут на Netlify: [https://yapweb-messenger.netlify.app/](https://yapweb-messenger.netlify.app/)

## Описание

Веб-приложение чата с следующим функционалом:

- **Авторизация** — вход по логину и паролю
- **Список чатов** — отображение контактов с последними сообщениями
- **Страница профиля** — просмотр и редактирование данных пользователя
- **Изменение пароля** — безопасная смена пароля
- **Загрузка аватара** — изменение фотографии профиля
- **Страницы ошибок** — 404 и 500

## 🛠 Технологии

- **Vite** — сборщик проектов
- **TypeScript** 
- **Handlebars** — шаблонизатор
- **SCSS** — препроцессор CSS
- **Netlify** — хостинг и автодеплой
- **Husky** — pre-commit хуки
- **ESLint** — линтинг кода
